ThreatLens Privacy Policy

Effective Date: April 15, 2026

🛡️ Our Commitment

ThreatLens is built to protect you online. We collect only the minimum data necessary for real-time threat detection. We do not sell your data under any circumstances.

1. Overview

ThreatLens analyzes websites you visit to detect phishing, malware, and unsafe pages. By using this extension, you agree to this policy.

2. What Data We Collect

Automatically Collected Data

Data Purpose
Domain (origin only) Used for threat scoring (no full URL stored)
HTTPS status Security indicator
Form / iframe / script counts Detect phishing patterns
Keyword count Identify suspicious content
Page title Displayed in extension UI

We DO NOT Collect

  • Full URLs (paths, query params)
  • Browsing history
  • Form inputs or passwords
  • Personal data or identity

3. How We Use Data

Data is securely sent over HTTPS to our analysis server:
https://threatlens-api.vercel.app/predict

The server returns a risk score and flags which are shown in the extension.

4. Third-Party Services

  • Vercel (API hosting)
  • Google Gemini API (explanations only, no personal data)

5. Data Storage

We do not store user data on servers. Temporary results may be cached locally in your browser.

6. Permissions

ThreatLens requires access to web pages to analyze threats in real time. This includes all HTTP/HTTPS sites.

7. Security

  • All communication is encrypted (HTTPS)
  • No credentials are collected
  • No tracking or fingerprinting

8. Your Control

You can disable or uninstall the extension anytime via Chrome settings.
